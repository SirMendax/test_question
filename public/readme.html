Тестовое задание на позицию PHP-разработчика

Задание

Начальные данные

1. Структура таблицы mysql
  • `services` - сервисы пользователей
  • `tarifs` - тарифы сервисов
  • Тарифы связываются между собой через параметр `tarif_group_id`
2. Файл с конфигом для доступа к БД - db_cfg.php

Задача: написать код, реализующий API-запросы

  • `GET /users/{user_id}/services/{service_id}/tarifs` - получает тарифы для конкретного сервиса. Тарифы могут быть только с тем же самым tarif_group_id, что и у текущего тарифа сервиса.
  • `PUT /users/{user_id}/services/{service_id}/tarif` Данные: `{"tarif_id": (ID тарифа из запроса на получениe тарифов)}`
